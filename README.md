# Тихо

Веб-приложение на **React** и **Vite**: лендинг, онбординг, экраны приложения (дашборд, трекер, дневник, друзья, библиотека, настройки).

## Запуск

```bash
npm install
npm run dev
```

Сборка: `npm run build`, предпросмотр билда: `npm run preview`.
